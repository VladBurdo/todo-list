## Installing Node.js

 - Open terminal
 - Type `npm install`

## Installing Bower Packages

`npm install` will install these too, but you can do it manually.

- Open terminal
- Type `npm start`

## Running

Runs locally, no database required.

## Running the Application
- browser to `http://localhost:8080/`
